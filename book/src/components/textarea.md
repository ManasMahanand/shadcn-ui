## Installation

<div class="warning">

The CLI is not yet available. For now, manually copy the component source into your project.

</div>

{{#tabs global="framework" }}
{{#tab name="Yew" }}

```shell
rust-shadcn-ui add textarea
```

-   [View source](https://github.com/RustForWeb/shadcn-ui/tree/main/packages/yew/textarea)

{{#endtab }}
{{#endtabs }}

## Usage

{{#tabs global="framework" }}
{{#tab name="Yew" }}

```rust,ignore
use yew::prelude::*;

use crate::components::ui::textarea::Textarea;

#[component]
fn Usage() -> impl IntoView {
    view! {
        <Textarea />
    }
}
```
